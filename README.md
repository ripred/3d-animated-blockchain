Interactive Three.js node-network simulation with configurable blockchain-style visuals.

Open `3d-animated-blockchain.html` in a browser to run the demo. The scene includes animated nodes, pulses, starfield controls, mouse interaction, and a generated-code panel for exporting the configured view.

## Notes

- The demo is a static HTML file.
- External browser dependencies are loaded from public CDNs.
- Settings are stored in browser local storage.
